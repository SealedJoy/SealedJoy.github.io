App & Linux

Systems, Terminals, Configs, Apps

Termux Expert Launcher is an open-source android application created to allow power-terminal users complete control of their device. It provides a minaturised linux distribution complete with its own package manager. It replaces the users homescreen with a terminal, and provides applets which lever the android API to interact with the devices core functionality.
