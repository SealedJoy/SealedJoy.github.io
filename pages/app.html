The TEL Mobile App

Termux Expert Launcher is a freely provided open-source android application. It was created to allow power users maximum control over their device. It provides a minaturised linux distribution complete with its own package manager. It replaces the users homescreen with a terminal, and provides applets which lever the android API to interact with the devices core functionality. It does not require root.

Working as a 2 person team who developed the TEL app, I undertook the development & configuration roles of the shell environment, installer, updater, setup files and documentation. I also created the python status manager, modules and applet binaries which provide the phone API interaction such as starting phone calls or launching an app.

I also created the character imagery & logos for the app and associated material, see the graphics page for a gif showing logo development.
